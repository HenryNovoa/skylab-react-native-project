# utils package

This packages contains different and reusable utils

## run tests

Inside the utils package root: 

```$ yarn run test```

Inside the project root: 

```$ yarn workspace @skylab/utils test```


## run lint

Inside the utils package root: 

```$ yarn run lint```

Inside the project root: 

```$ yarn workspace @skylab/utils lint```


### Contains:

- Assert Utils
- Date Utils
